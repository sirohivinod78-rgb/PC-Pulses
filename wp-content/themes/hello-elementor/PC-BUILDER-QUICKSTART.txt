3-MINUTE SETUP
════════════════════════════════════════════════════════════════════════

STEP 1: Create the WordPress Page
   → Go to WordPress Admin Dashboard
   → Click "Pages" → "Add New"
   → Title: "Build Your PC"
   → Scroll down to "Template"
   → Select "PC Builder" from dropdown
   → Click "Publish"
   → Note the page URL

STEP 2: Update the Button Link
   → Edit any page with the "Build Your PC" button
   → Click on the button to edit
   → Change link to the PC Builder page URL from Step 1
   → Save/Publish

STEP 3: Test It!
   → Click the "Build Your PC" button
   → Select a component (CPU, GPU, etc.)
   → Watch the compatibility checking work
   → Try exporting your build

WHAT HAPPENS WHEN USER CLICKS "BUILD YOUR PC"
════════════════════════════════════════════════════════════════════════

1. User clicks button → Goes to PC Builder page
2. Sees component grid with categories (CPU, GPU, Motherboard, etc.)
3. Selects first component (e.g., CPU)
   → Card highlights
   → Build summary updates
   → Compatibility checks run
4. Selects second component (e.g., Motherboard)
   → Validates socket compatibility
   → Shows warning if mismatch
   → Updates total price
5. Continues selecting components
   → Real-time updates
   → Automatic compatibility checks
   → Progress bar fills as components selected
6. Can remove components by clicking "Remove"
7. Exports build as JSON file (📥 Export Build button)
8. Resets everything (🔄 Reset Build button)

COMPATIBILITY RULES IMPLEMENTED
════════════════════════════════════════════════════════════════════════

✓ CPU Socket → Motherboard Socket
  ERROR if mismatch (e.g., LGA1700 CPU on AM5 motherboard)

✓ RAM Type → Motherboard RAM Type
  ERROR if mismatch (e.g., DDR5 RAM on DDR4 motherboard)

✓ Total Power Consumption → PSU Wattage
  WARNING if PSU < (CPU TDP + GPU Power + 150W) × 1.2
  (20% headroom recommended for stability)

✓ CPU Socket → Cooler Socket Support
  ERROR if cooler doesn't support CPU socket

TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════

Problem: Page shows "Template not found"
Solution: 
  - Ensure template-pc-builder.php is in hello-elementor folder
  - Refresh WordPress page template cache
  - Try selecting template again

Problem: Components not displaying
Solution:
  - Check browser console for JavaScript errors (F12)
  - Verify pc-builder.js is loading (Network tab)
  - Check that pc-builder-data.php has valid PHP syntax

Problem: Prices not showing correctly
Solution:
  - Check pc-builder-data.php for price values
  - Verify currency is USD ($ symbol)
  - Clear browser cache (Ctrl+Shift+Delete)

Problem: Compatibility warnings not appearing
Solution:
  - Ensure both components have required fields
  - Check browser console for errors
  - Verify component data structure in pc-builder-data.php

Problem: Export not working
Solution:
  - Check browser console for errors
  - Ensure JavaScript functions are defined
  - Test in different browser
  - Check file permissions
